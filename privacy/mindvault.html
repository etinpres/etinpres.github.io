MindVault 개인정보처리방침

시행일: 2026년 4월 11일

1. 수집하는 개인정보

MindVault는 개인정보를 수집하지 않습니다.

모든 지식 베이스 데이터(그래프, 위키, 검색 인덱스)는 사용자의 로컬 파일 시스템에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 개인정보의 이용 목적

수집하는 개인정보가 없으므로 해당 사항이 없습니다.

3. 개인정보의 보유 및 파기

모든 데이터는 사용자 로컬 환경에만 존재하며, 사용자가 직접 삭제할 수 있습니다.

4. 제3자 제공

개인정보를 제3자에게 제공하지 않습니다.

5. 외부 API 사용

시맨틱 추출 기능 사용 시 사용자가 직접 설정한 LLM API(로컬 또는 클라우드)를 호출할 수 있습니다. 이 경우 사용자의 코드 및 문서 내용이 해당 API 제공자에게 전송될 수 있으며, 각 제공자의 개인정보처리방침이 적용됩니다. 이 기능은 사용자의 명시적 동의 후에만 활성화됩니다.

6. 문의

개인정보 관련 문의: Dr.Ocean@gmail.com

7. 변경 사항

본 방침이 변경될 경우 본 페이지를 통해 안내합니다.
